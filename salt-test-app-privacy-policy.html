Facebook SALT Test App Privacy Policy

This test application "SALT Test App" is only used by the creator(s) for testing out the Facebook Graph API. No information from any other users will be stored.

This application may request, retrieve and store access tokens and long lived access tokens for the purpose of testing out the Facebook Graph API. Uses of the tokens will be according to the user authorized permission scopes. This may include but not be limited to basic profile information, ability to post and retrieve posts, stories and other information relating to Facebook Pages.
